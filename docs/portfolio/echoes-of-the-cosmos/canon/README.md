# Canon

This folder contains the public source-canon draft for Echoes of the Cosmos.

## Files

- `ECHOES_OF_THE_COSMOS.md` - main canon reference guide.
- `THE_RESONANCE_Magic_System.md` - magic/system reference for Resonance.
- `CANON_INDEX.md` - short live-canon anchor and contribution policy.

## Policy

All core canon is intended to be public unless a file is explicitly marked private.

Generated drafts, private memories, and old test-universe outputs are not canon by default.
